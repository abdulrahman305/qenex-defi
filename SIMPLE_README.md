## 💻 Manual Setup (3 Steps)

### Step 1: Clone
```bash
git clone https://github.com/abdulrahman305/qxc-token.git
cd qxc-token
```

### Step 2: Install
```bash
npm install
```

### Step 3: Run
```bash
npm start
```

Open https://abdulrahman305.github.io/qenex-docs in your browser.
