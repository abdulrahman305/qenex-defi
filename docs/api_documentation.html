  "api_key": "qnx_aBcD1234..."

Pipelines

GET /api/pipelines
List all pipelines
limit integer Default: 10
status string Filter by status: pending, running, success, failed
Response: 
[
  {
    "id": "pipeline_20240120123456",
    "name": "main-build",
    "repository": "qenex/app",
    "branch": "main",
    "status": "running",
    "created_at": "2024-01-20T12:34:56Z"
  }
]
POST /api/pipelines/trigger

repository string required
branch string Default: main
commit string

  "message": "Pipeline triggered successfully"
}

Rate Limiting

API requests are limited to:

  • 1000 requests per hour for authenticated users
  • 100 requests per hour for unauthenticated requests

Error Responses

All error responses follow this format:

{
  "error": "Error message",
  "code": "ERROR_CODE",
  "details": {}
}

WebSocket Events

Connect to ws://localhost:8000/ws for real-time updates:

  • pipeline:started - Pipeline execution started
  • pipeline:completed - Pipeline execution completed
  • deployment:success - Deployment successful
  • alert:triggered - System alert triggered
