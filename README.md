## Quick Start

```bash
# Clone repository
git clone https://github.com/abdulrahman305/qenex-defi.git
cd qenex-defi

# Install dependencies
pip install -r requirements.txt
npm install

# Deploy contracts (testnet)
npx hardhat run scripts/deploy.js --network goerli

# Start DeFi engine
python defi_engine.py
```
